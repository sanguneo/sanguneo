### 🧑‍💻 About me

- 🛠️ **Frontend-leaning full-stack** engineer — TypeScript / React / Node 를 주력으로 씁니다.
- 🤖 요즘은 **AI 에이전트 · MCP · 테스트 자동화 · 개발 생산성 도구**를 많이 만듭니다.
